# SkillSwap

## Purpose
SkillSwap is a React + Vite web app to connect learners with instructors to exchange skills and discover classes.

## Live URL


## Key Features
- Browse skill listings and detailed pages
- Instructor profiles and top instructors section
- Authentication (register, login, forgot password)
- Protected routes for membership/profile areas
- Responsive UI with TailwindCSS + DaisyUI and animated sliders
- Firebase-powered backend (auth and data)

## npm packages used


- react, react-dom — UI library
- vite — dev server / bundler
- tailwindcss, daisyui — styling and UI components
- @tailwindcss/vite — Tailwind integration for Vite
- react-router, react-router-dom — client-side routing
- firebase — backend services (auth, database, hosting)
- swiper — image/content sliders
- aos — scroll animations
- lucide-react — icon set
- react-hot-toast — toast notifications
